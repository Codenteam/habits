## Default Permission

Default permissions for Local AI plugin - allows all AI operations

#### This default permission set includes the following:

- `allow-generate-text`
- `allow-image-caption`
- `allow-image-gen`
- `allow-transcribe`
- `allow-text-to-voice`
- `allow-embed`
- `allow-utilities`
