An AI-powered recipe generator that creates personalized recipes based on your available ingredients, dietary restrictions, and cuisine preferences—complete with AI-generated images, shopping lists, and nutritional info.

## What It Does

- **Generate Recipes**: Input your ingredients and get a complete recipe with step-by-step instructions
- **AI-Generated Images**: Beautiful food photography generated for each recipe
- **Smart Shopping Lists**: Identifies missing ingredients and suggests substitutions
- **Nutritional Analysis**: Estimates calories, macros, and dietary information
- **Recipe Storage**: Saves recipes to a database for later access

## API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/generate-recipe` | POST | Generate a new recipe from ingredients |
| `/api/list-recipes` | GET | List all saved recipes |
| `/api/get-recipe` | GET | Get a specific recipe by ID |

## Quick Start

<ExampleRunner examplePath="ai-cookbook" />

::: code-group
```bash [Generate Recipe]
curl -X POST http://localhost:13000/api/generate-recipe \
  -H "Content-Type: application/json" \
  -d '{
    "input": {
      "ingredients": "chicken breast, garlic, olive oil, lemon, rosemary",
      "restrictions": "gluten-free",
      "cuisine": "Mediterranean",
      "mealType": "dinner",
      "servings": 4
    }
  }'
```

```bash [List Recipes]
curl http://localhost:13000/api/list-recipes
```

```bash [Get Recipe]
curl "http://localhost:13000/api/get-recipe?input[id]=YOUR_RECIPE_ID"
```
:::

## Key Features

### Intelligent Recipe Generation

The workflow uses GPT-4o to create creative, practical recipes that make the most of your available ingredients:

- Considers dietary restrictions (vegetarian, vegan, gluten-free, etc.)
- Adapts to preferred cuisine types
- Scales for different serving sizes
- Includes chef tips and variations

### AI-Generated Food Photography

Each recipe comes with a professionally-styled food image generated by DALL-E 3.

### Smart Shopping Assistant

Compares your available ingredients with recipe requirements and provides:
- Missing ingredients list with quantities
- Optional ingredients for extra flavor
- Substitution suggestions

### Nutritional Information

Estimates per-serving nutritional data including:
- Calories, protein, carbs, fat, fiber
- Health notes and dietary classifications

## Workflow Architecture

The `generate-recipe` workflow demonstrates parallel AI processing:

```
[Input] → [Generate Recipe] → [Generate Image]    ↘
                           → [Shopping List]      → [Save to DB] → [Output]
                           → [Nutrition Analysis] ↗
```

1. **Generate Recipe**: Creates the main recipe JSON with ingredients and instructions
2. **Parallel Processing**: Simultaneously generates image, shopping list, and nutrition info
3. **Save**: Stores everything in the database
4. **Return**: Outputs the complete recipe package
