# Node Types

Habits supports three types of nodes for building automation workflows.

## 1. Bits

Native Habits nodes with three flavors:

- **Declarative** – JSON-based configuration, no code required
- **Script** – Inline JavaScript/TypeScript logic
- **Programmatic** – Full TypeScript modules for complex operations

## 2. ActivePieces

Import nodes from [ActivePieces](https://activepieces.com) ecosystem. Hundreds of pre-built integrations available, mostly MIT licensed. 

## 3. n8n

Import nodes from [n8n](https://n8n.io) ecosystem. Extensive community-built integrations, most nodes are open-source but the runtime which is needed to run the nodes is SUL, so use with cautious.    

---

::: tip Mix and Match
You can combine all three node types in a single workflow.
:::
