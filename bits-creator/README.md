### What Changed?

The `bits-creator` package is no longer maintained as a standalone module. All of its features and capabilities have been **fully integrated** into the main [`habits base`](../packages/habits/base) package. This consolidation streamlines development, reduces duplication, and ensures all users benefit from the latest updates and improvements in a single place.

### What Should You Do?

- **Migrate:**  
  If you previously used `bits-creator`, switch to using `habits base` for all related workflows and development tasks.
- **Update Scripts:**  
  Update any scripts, CI/CD pipelines, or documentation that reference `bits-creator` to use `habits base` instead.
- **Unified Documentation:**  
  All documentation and examples are now maintained under the main `habits base` package. Refer to the [main README](../packages/habits/README.md) for up-to-date usage instructions.

### Need Help?

If you encounter any issues migrating or have questions about the new workflow, please open an issue in the main repository or reach out to the maintainers.
