## Default Permission

Default permissions for the Matter smart home plugin

#### This default permission set includes the following:

- `allow-discover-devices`
- `allow-get-devices`
- `allow-get-device-state`
- `allow-set-on-off`
- `allow-set-level`
- `allow-set-color`
- `allow-commission-device`
- `allow-remove-device`
