## Default Permission

Default permissions for the system-settings plugin

#### This default permission set includes the following:

- `allow-get-volume`
- `allow-set-volume`
- `allow-set-mute`
- `allow-get-ringer-mode`
- `allow-set-ringer-mode`
- `allow-get-bluetooth-state`
- `allow-set-bluetooth`
- `allow-get-dnd-state`
- `allow-set-dnd`
