# Permissions

This plugin has the following permission scopes:

## `allow-send-sms`

Allows sending SMS messages.

## `allow-read-sms`

Allows reading SMS messages (Android only).

## `deny-send-sms`

Denies sending SMS messages.

## `deny-read-sms`

Denies reading SMS messages.

## Default Permissions

The default set of permissions allows sending and reading SMS:

- `allow-send-sms`
- `allow-read-sms`
