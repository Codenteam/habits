## Default Permission

Default permissions for the email plugin

#### This default permission set includes the following:

- `allow-send-email`
- `allow-connect-imap`
- `allow-disconnect-imap`
- `allow-list-mailboxes`
- `allow-select-mailbox`
- `allow-fetch-emails`
- `allow-fetch-email-by-uid`
- `allow-search-emails`
